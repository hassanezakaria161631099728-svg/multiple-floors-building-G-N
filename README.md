# multiple-floors-building-G-N
this is an open source program to calculate the general case of a multiple story building G+N 
enter these inputs:
number of floors: n_floors
height of the floor: h
number of horizontal spans: n_spans
lenght of the single horizontal span: L
note that the floor height has to be the same value on each story and same for span lenght L
then you just execute the main.cpp file
adjust just he inputs according to your building variante
CPP apps: this folder is where I stacked various main.cpp files as you can distiguish them from functions by 
having a loop which starts with int main and ends with return 0; 
so to execute one them you have to put it at the same level as functions folder 
 like this
  functions
  main.cpp
 you can't run it inside cpp apps folder along side with many main.cpp files
 I have specified each one so you don't get confused
